# Barbell Calculator

A simple barbell plate calculator built as a single-page HTML app.

This is a vibe coded project. 

## Features

- Support for men's (20 kg) and women's (15 kg) barbells
- 10 standard Olympic plate sizes from 25 kg down to 0.5 kg
- Visual barbell diagram that updates as plates are added
- Plate list sorted smallest to largest
- Undo and clear actions
- Color-coded plates following IWF competition standards

## Usage

Open `index.html` in a browser.
